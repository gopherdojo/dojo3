# 課題3-1
- 標準出力に英単語を出す（出すものは自由）
- 標準入力から1行受け取る
- 制限時間内に何問解けたか表示する

## 基本設計
- main
    - 入出力パッケージ if
        - 出題と採点パッケージ dealer
            - out if
            - in if
            - from 単語パッケージ words
            - with 制限時間パッケージ clock
            - with 残機システム lives
            - to 採点集計パッケージ scorer
            - to ランキングパッケージ ranking

## チャネルで繋げては？（チャネル練習）
game chan
    word 単語を選ぶ
    question 出題する
    listener 回答を聞く
    checker 答えあわせをする
    scorer 採点する
    timer 時間を計測する
